# RxKakaoSDKAuth

RxSwift를 사용하는 사용자 인증 및 토큰 관리 모듈입니다.

## Requirements
- Xcode 11.0
- iOS 11.0
- Swift 5.0
- CocoaPods 1.8.0

## Dependencies
- KakaoSDKAuth
- RxKakaoSDKCommon

## Installation
```
pod 'RxKakaoSDKAuth'
```

## Import
```
import RxKakaoSDKAuth
```

## Usage
[AuthController](Extensions/Reactive.html) 클래스를 이용하여 카카오 로그인을 적용할 수 있습니다.
